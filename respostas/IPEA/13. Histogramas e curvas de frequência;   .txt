Item do edital: 13. Histogramas e curvas de frequência;   
 
Histogramas e curvas de frequência são ferramentas estatísticas utilizadas para representar a distribuição de dados em um conjunto de observações. Vamos entender cada um deles em detalhes:

1. Histograma:
   - Um histograma é um gráfico de barras que representa a distribuição de frequência de um conjunto de dados contínuos ou discretos.
   - No eixo horizontal (eixo x), temos os intervalos ou classes dos dados, e no eixo vertical (eixo y), temos a frequência ou densidade de frequência.
   - As barras do histograma são desenhadas adjacentes umas às outras, sem espaços entre elas, para representar a continuidade dos dados.
   - A altura de cada barra representa a frequência ou a densidade de frequência dos dados dentro da classe correspondente.

2. Curva de Frequência:
   - Uma curva de frequência é uma representação gráfica suave da distribuição de frequência de um conjunto de dados contínuos.
   - É obtida conectando os pontos médios superiores de cada barra do histograma por meio de uma linha suave.
   - A curva de frequência é útil para visualizar a forma geral da distribuição dos dados e identificar padrões, como simetria, assimetria, picos, entre outros.

Diferenças entre Histogramas e Curvas de Frequência:
- O histograma é composto por barras retangulares adjacentes, enquanto a curva de frequência é uma linha suave que conecta os pontos médios superiores das barras.
- O histograma é mais adequado para dados discretos ou agrupados em classes, enquanto a curva de frequência é mais útil para dados contínuos.
- O histograma fornece uma representação mais detalhada da distribuição de frequência, enquanto a curva de frequência é mais útil para visualizar a forma geral da distribuição.

Em resumo, histogramas e curvas de frequência são ferramentas essenciais para analisar e visualizar a distribuição de dados, permitindo uma compreensão mais profunda das características estatísticas de um conjunto de observações.
